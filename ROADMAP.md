# 🗺️ Roadmap - Zen Garden

## v0.1.0 - MVP (Current) ✅
- [x] Jardín zen básico en 3D
- [x] Rastrillo interactivo para dibujar líneas
- [x] Sonido ambiente blanco relajante
- [x] Sonidos ocasionales tipo "bong"
- [x] Mariposa animada
- [x] Controles de cámara suave
- [x] Interfaz limpia y minimalista
- [x] Configuración para GitHub
- [x] Configuración para Netlify
- [x] Documentación de despliegue

## v0.2.0 - Elementos Interactivos
- [ ] Sistema de colocación de piedras mejorado
- [ ] Diferentes formas y tamaños de piedras
- [ ] Poder mover y eliminar elementos
- [ ] Sistema de guardado local (localStorage)
- [ ] Poder resetear el jardín
- [ ] Animación de ondas en arena

## v0.3.0 - Nuevos Elementos
- [ ] Árboles decorativos
- [ ] Agua (pequeño estanque)
- [ ] Puente de madera
- [ ] Linterna/Luz asiática
- [ ] Flores
- [ ] Diferentes tipos de mariposas

## v0.4.0 - Sonido Mejorado
- [ ] Más tipos de bongs con diferentes tonos
- [ ] Sonidos ambientes variados (viento, agua)
- [ ] Sonidos de reacción al interactuar
- [ ] Visualizador de audio
- [ ] Control de volumen granular

## v0.5.0 - Interfaz y UX
- [ ] Modo oscuro/claro
- [ ] Selector de temas
- [ ] Paleta de colores personalizable
- [ ] Tutorial interactivo
- [ ] Animación de introducción suave
- [ ] Hints/tooltips

## v0.6.0 - Persistencia (Neon Database)
- [ ] Backend Node.js/Express
- [ ] Integración con Neon PostgreSQL
- [ ] Autenticación básica (opcional)
- [ ] Guardar y cargar diseños
- [ ] Galerería de diseños públicos
- [ ] Sistema de favoritos

## v0.7.0 - Secciones Temáticas
- [ ] Acceso desde el mapa a diferentes escenas
