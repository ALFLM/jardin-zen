# 🧘 Zen Garden - A Meditative Experience

Un jardín zen interactivo y tranquilo hecho con React, Three.js y Web Audio API.

## 🌿 Características

- **Rastrillo interactivo**: Usa tu ratón para hacer líneas relajantes en la arena
- **Piedras decorativas**: Coloca piedras en el jardín (próximamente interactivo)
- **Mariposa animada**: Una mariposa que vuela suavemente sobre el jardín
- **Sonido ambiente zen**: Sonido blanco relajante con ocasionales "bongs" meditativo
- **Vista isométrica 3D**: Experiencia visual inmersiva con Three.js
- **Controles suave**: Zoom y rotación fluida de la cámara

## 🎮 Controles

| Acción | Control |
|--------|---------|
| Rastrillo | Arrastra el ratón |
| Zoom | Rueda del ratón |
| Rotación | Click derecho + Arrastrar |
| Sonido | Botón inferior izquierda |
| Información | Botón superior derecha (?) |

## 🚀 Instalación Local

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/zen-garden.git
cd zen-garden

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Crear build de producción
npm run build

# Vista previa del build
npm run preview
```

## 📦 Tech Stack

- **React 18** - Librería UI
- **Vite** - Build tool rápido
- **Three.js** - Renderizado 3D
- **Web Audio API** - Sonido ambiente y efectos
